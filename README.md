# Что это такое:
Этот плагин внутри документа CorelDraw добавляет метки и штрихкоды для плоттеров SummaCut, выводит файл резки в формате plt в указанную пользователем папку.
Также имеется небольшой набор настроек, которые можно отправить в плоттер внутри задания резки:
<p align="center">
<img width="314" height="532" alt="image" src="https://github.com/user-attachments/assets/80b94f41-3fc2-452c-ad01-4d9660314989" />
</p>
После установки нужно перезапустить CorelDraw, если он был запущен во время установки, установщик необходимо запускать от имени администратора, иначе библиотека не зарегистрируется в системе и вызов класса библиотеки через vba макрос будет невозможен.
В кореле появится новая панель управления, вызвать которую можно будет через отдельный пункт нового меню, на панели всего две кнопки:

<p align="center">
<img width="203" height="59" alt="image" src="https://github.com/user-attachments/assets/53526703-3716-479d-ae5a-f3961b2face4" />
</p>

При нажатии "Поставить метки" плагин добавит метки и баркоды, создаст рабочие слои и выведет plt файл резки.
